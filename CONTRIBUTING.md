# Contributing to Open Rec Sphere


Thank you for considering contributing! This project is both a learning lab and a production-style portfolio project. Whether you want to fix a typo or design a new candidate generation module, you're welcome here.

## How to Contribute

1. **Fork the repository**
2. **Create a feature branch**
   `git checkout -b feature/your-feature-name`
3. **Make your changes**
4. **Write tests** (if applicable)
5. **Commit and push**
   `git commit -m "Describe your changes"`
   `git push origin feature/your-feature-name`
6. **Open a pull request** — link it to an open issue if possible.

## Code Style

- Python: PEP8
- Use type hints where possible
- Keep modularity and readability in mind

## Questions?

Open a GitHub Discussion or start a thread in Issues.

Thanks again!
